git安装

gitbash打开,cd到想要想要创建博客的目录下

hexo搭播客

把hexo博客放到github上面

ssh把本地和github连接

gitbash输入命令生成密钥

ssh-keygen -t ed25519 -C “your_email@example.com“

生成的密钥在本地C:\Users\Huawei.ssh文件夹

在github设置里面把本地密钥添加进去

可使用如下命令验证上述配置是否成功

ssh -T git@github.com

需要在生成的hexo博客文件下面修改配置文件_config.yml

deploy:
type: git
#你的repositiry名字也就是你的github名字
repository: git@github.com:wudidekbl/wudidekbl.github.io.git
branch: main
