发表于 2024-08-01 更新于 2024-12-10
本文字数： 1.9k 阅读时长 ≈ 4 分钟

yum install 失败

确保通外网

看看能不能ping通8.8.8.8 或者ping www.baidu.com

设置DNS服务器

vim /etc/resolv.conf

nameserver 8.8.8.8

nameserver 114.114.114.114

如果还是yum install 失败

换yum源

mkdir /etc/yum.repo.d/bak/

mv /etc/yum.repos.d/Centos-* /etc/yum.repo.d/bak/

vi /etc/yum.repos.d/CentOS-Base.repo
