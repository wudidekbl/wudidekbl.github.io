配置服务

创建KEY，KEY要复制出来保存：Setup Keys -> Create Setup Key

主平台、分平台各自注册一个

Auto-assigned groups创建命名方式：输入名称后回车创建

主平台：渠道名称-M

分平台：渠道名称-S

添加访问策略：Access Control -> Policies -> Add Policy，命名为渠道名称

注册认证服务，在主平台、分平台服务器上（KEY不能重复用于其他平台，只针对当前平台使用）

安装xsetnet服务

主平台注册：xsetnet up –setup-key 0D7E2FF0-964B-40A2-BFBC-017094586D8F

分平台注册：xsetnet up –setup-key A1061841-8A4D-45E2-B705-00943BA8E8CF

注册成功会返回Connected

注册都成功后，在netbird平台上查看 -> Peers
