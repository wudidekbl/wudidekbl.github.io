- vim 是vi的升级版，是linux内置的文本编辑器

- vim hello.java 会创建一个文件和进入这个文件，按i开始插入，按esc是退出插入，退出插入按:  输入wq是写入保存并退出文件，q是退出文件

- xshell关机重启命令

- shutdown
    - shutdown -h now 立刻关机
    - shutdown -h 1 一分钟后关机
    - shutdown -r now 立即重启
- halt 关机
- reboot 重启
- syn 吧内存的数据存到磁盘，注意，在关机之前执行syn防止没有保存的数据消失
- logout用户注销，不在占有这个连接

- linux用户管理

- useradd  【 用户名】 添加用户
- useradd -d 目录名  用户名  指定到用户创建到哪个目录下
- su 用户名  切换用户，高权限切换到低权限不需要输入密码，
- whoami 查看当前用户是谁 ![图片](./linux基础篇-幕布图片-711177-939205.jpg)
- cd 目录名 change dir
- passwd 用户名    给用户修改创建密码 root下
- userdel 用户名 删除用户  root下
- userdel -r 用户名 删除用户同时和他的家目录
- id 用户名 ![图片](./linux基础篇-幕布图片-44732-615196.jpg)

- 用户组的管理

- 类似与角色，系统可以对有共性的用户进行管理
- groupadd  组名
- groupdel 组名 删除组
- useradd -g 用户组 用户名 创建用户和指定他的组 ![图片](./linux基础篇-幕布图片-666466-73203.jpg)
- usermod -g 用户组 用户名 修改用户的组

- 用户和组的的配置文件

- 用户配置文件,/etc/passwd(用户信息）依次是：用户名，用户密码，用户id，组id，家目录，对应shell解释器 ![图片](./linux基础篇-幕布图片-716772-786888.jpg)
- 组配置文件，（组的信息）/etc/group
- 口令配置文件（密码和登录信息 ，是加密的）/etc/shadow

- 帮助指令

- man 命令或者配置文件 man ls
- help 命令  help cd

- 文件目录类指令

- ls 显示文件 -l 列表形式显示加上权限信息，-a所有的都显示，包括隐藏的
- cd 回到自己家目录，cd  ../回到上一级
- mkdir 指令用于创建目录
- mkdir -p 可以创建多级目录不存在的多级目录，
- vim 文件名 创建并进入这个文件
- touch 文件名  创建一个文件，但是不进入他（一次性创建多个文件可以）
- cp 选项  source  dest   拷贝指定文件到指定目录
- cp -r  可以复制整个目录过去 \cp -r 强制覆盖不会提示有相同文件了
- pwd 显示当前位置在哪个目录
- 上下键可以调出之前使用的指令
- rm移除文件或目录 -r 递归删除，-f 强制删除不提示
- rmdir 删除空目录，不能删目录里面有内容的目录
- rm -rf  可以删除目录里面有内容的目录
- mv 移动文件，与目录 或重命名 ，如果就是在当前目录中就是重命名
- cat 以只读的方式查看文件，不能修改-n 显示行号cat  -n 文件 | more 以全屏的方式按页显示文本文件内容
- more 指令 查看文件
- less 用来查看较大的文件，先加载一份分显示
- 指令>输出重定向 和>>追加
- > ls-l >a.tex 把列表的内容写入文件a.txt中（覆盖） ls -al >>a.txt 把列表i内容追加到a.txt的末尾 文件1>文件二 把文件1内容写入到文件2（覆盖 echo “字符” >> 文件1 把字符追加到文件1中
- echo 输入内容，输出内容到控制台。echo $PATH echo “hellowordl”
- head -n 5 文件 显示文件前5行
- tail -n 5 显示文件尾部后5行 tail -f 文件 实时追踪该文档的更新
- ln -s 源文件或者目录  软连接名  ln -s  /root  linkroot   cd linkroot就会到root目录，rm -rf linkroot 删除软链接
- history指令，查看已经执行的指令的历史  ;   !15执行历史编号为15的指令

- date 显示当前时间
- date +%Y
- date +%m

- date -s +2020-11-1+11:22:10设置时间和日期
- cal以日历的方式显示 cal 2020 显示2020的1-12月日历 ![图片](./linux基础篇-幕布图片-731903-478896.jpg)

- 查找搜索类指令

- find 查找范围 选项 文件   find /home -name hello.java
- -user  hww按文件用户
- find /  -size  +20 查找大于20m的文件 -20就是小于20m的
- locate指令 快速定位文件路径，第一次远行locate必须要使用updatedb指令创建locate数据库
- grep指令过滤查找，管道符|  表示将前一个命令的处理结果输出传递给后面的命令处理
- 显示hello.txt 里面的内容里的yes出来 cat helllo.txt | grep -n yes

- 压缩解压指令

- gunzip 解压
- gzip压缩 使用gzip压缩，不会保留原来的文件，原来的文件变成压缩文件
- unzip -d  想要压缩到的目录 想要压缩的文件； 指定解压后的文件放在哪里
- zip -r 取得解压名  想要压缩的文件；zip后缀
- tar既能解压又能压缩 tar后缀

- 权限的管理

- 第一 个 -是普通文件，d是目录文件，c是设备文件
- 第二组 3个 rwx 所有者权限，第二组 rwx所有组权限，第三组rwx 其他组权限后面的数字是 文件是硬链接的个数，目录是子目录的个数
- chmod u=rwx，g=rwx ，o=rwx  文件名字 修改文件权限
- chmod u+w g-x
- r=4 w=2 x=1
- chmod 755 abc.txt
- chown newowner 文件名  改变文件所有者
- chown newowner:newgroup 文件名 改变文件所有者和所有组 -R如果是目录，则目录下的子文件和目录都生效
- chown -R tom kkk  将kkk目录下的子文件的所有者都改成tom
- 如果一个目录的权限没有开放给其他组，那么别的用户就无法cd到该目录也要开启x执行权限不然无法执行cd指令
